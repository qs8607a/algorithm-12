algorithm
=========

some algorithm test

To build this project:
-  svn co https://github.com/snnn/algorithm/trunk algorithm
-  cd algorithm
-  mkdir b1
   cd b1
   cmake ..
-  make
   make test


- src/SkipList.h : A SkipList.
- src/fib.cpp: 以非递归的方式计算fib数列。
- src/heap.h: 数据结构，堆。
- src/hash.cpp: 一个简单的hash函数。 char[] -> uint32_t
- src/randomImpl1.h: LCG随机数生成器。
- src/sort.h： 几种排序算法
